## Огляд

Регулярна зміна паролю підвищує безпеку вашого облікового запису. Procivo дозволяє змінити пароль у будь-який момент через сторінку профілю. Для зміни необхідно знати поточний пароль.

## Кроки

1. Перейдіть на сторінку **профілю** (іконка профілю у topbar → «Профіль»).
2. Знайдіть секцію **«Зміна паролю»**.
3. Заповніть форму:
   - **Поточний пароль** — введіть ваш діючий пароль для підтвердження особи.
   - **Новий пароль** — введіть новий пароль (мінімум 8 символів).
   - **Підтвердження нового паролю** — повторіть новий пароль.
4. Натисніть кнопку **«Змінити пароль»**.
5. Після успішної зміни ви побачите повідомлення про успіх.

### Вимоги до паролю

- Мінімальна довжина: **8 символів**.
- Рекомендовано: великі та малі літери, цифри, спеціальні символи.
- Пароль не може збігатися з попереднім.

## Поради

- Змінюйте пароль кожні 3-6 місяців для підвищення безпеки.
- Не використовуйте один і той самий пароль для різних сервісів.
- Після зміни паролю поточна сесія залишається активною — повторний вхід не потрібен.
- Якщо ви забули поточний пароль, зверніться до адміністратора організації.
