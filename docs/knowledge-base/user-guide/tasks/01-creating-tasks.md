## Огляд

Завдання — це основна одиниця роботи в Procivo. Кожне завдання має заголовок, опис, виконавця, пріоритет та опціональний дедлайн. Завдання можна створювати вручну або вони створюються автоматично при виконанні бізнес-процесів.

## Кроки

### Створення завдання

1. Перейдіть до розділу **«Завдання»** у бічній панелі.
2. Натисніть кнопку **«Створити завдання»** у верхній частині сторінки.
3. Заповніть форму створення:
   - **Назва** — короткий, зрозумілий заголовок завдання.
   - **Опис** — детальний опис того, що потрібно зробити.
   - **Виконавець** — оберіть відповідального з учасників організації.
   - **Пріоритет** — оберіть рівень: критичний, високий, середній або низький.
   - **Дедлайн** — встановіть дату та час, до яких завдання має бути виконано.
   - **Мітки** — додайте мітки для категоризації (опціонально).
4. Натисніть **«Створити»**.
5. Завдання з'явиться у списку зі статусом «Нове».

### Сторінка списку завдань

1. Відображає всі завдання організації у вигляді таблиці.
2. Підтримує фільтрацію за статусом, пріоритетом, виконавцем та мітками.
3. Підтримує сортування за датою створення, дедлайном та пріоритетом.
4. Пагінація для навігації по великих списках.

### Деталі завдання

1. Натисніть на завдання у списку для переходу до сторінки деталей.
2. На сторінці деталей:
   - **Основна область** — назва, опис, коментарі.
   - **Бічна панель** — статус, пріоритет, виконавець, мітки, дедлайн, дата створення.
3. Ви можете редагувати будь-яке поле прямо на сторінці деталей.

## Поради

- Давайте завданням зрозумілі назви — вони відображаються на канбан-дошках та в сповіщеннях.
- Завдання без дедлайну не з'являться у секції «Мої завдання» на дашборді.
- Пріоритет «Критичний» виділяється червоним кольором для привернення уваги.
