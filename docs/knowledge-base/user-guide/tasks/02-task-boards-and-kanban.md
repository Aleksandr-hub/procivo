## Огляд

Канбан-дошка — це візуальний інструмент для управління завданнями. Колонки дошки відповідають статусам завдань, а картки можна перетягувати між колонками для зміни статусу. Дошка також підтримує WIP-ліміти для контролю навантаження.

## Кроки

### Створення дошки

1. Перейдіть до розділу **«Дошки»** у бічній панелі.
2. Натисніть **«Створити дошку»**.
3. Оберіть тип **«Дошка завдань»** (Task Board).
4. Введіть назву дошки.
5. Натисніть **«Створити»**.

### Налаштування колонок

1. На створеній дошці натисніть **іконку налаштувань**.
2. Кожна колонка прив'язана до одного або кількох статусів завдань.
3. Налаштуйте порядок колонок перетягуванням.
4. Встановіть **WIP-ліміт** (Work In Progress) для обмеження кількості завдань у колонці:
   - При досягненні **80%** ліміту заголовок колонки стає **помаранчевим** (попередження).
   - При досягненні **100%** — **червоним** (перевищення).

### Робота з дошкою

1. Завдання відображаються як **картки** у відповідних колонках.
2. **Перетягніть** картку в іншу колонку для зміни статусу завдання (drag-and-drop).
3. На картці відображається: аватар виконавця, чіп пріоритету, мітки, дедлайн, кількість коментарів.
4. Натисніть на картку для переходу до деталей завдання.

## Поради

- WIP-ліміти допомагають уникнути перевантаження — команда зосереджується на завершенні поточних завдань перед початком нових.
- Використовуйте фільтри та свімлейни для фокусування на конкретних завданнях.
- Зміна статусу через drag-and-drop запускає ті ж самі бізнес-правила, що й зміна через API.
