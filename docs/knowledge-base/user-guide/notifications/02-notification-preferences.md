## Огляд

Procivo дозволяє налаштувати, які сповіщення ви отримуєте та через які канали. Ви можете окремо керувати сповіщеннями в додатку (in-app) та по електронній пошті (email) для кожного типу події.

## Кроки

### Доступ до налаштувань

1. Перейдіть на сторінку **профілю** (іконка профілю у topbar → «Профіль»).
2. Знайдіть секцію **«Налаштування сповіщень»** або перейдіть на окрему сторінку уподобань.

### Типи подій

Сповіщення доступні для таких типів подій:

| Тип події | Опис |
|-----------|------|
| Призначення завдання | Вам призначено нове завдання |
| Зміна статусу завдання | Завдання, за яким ви слідкуєте, змінило статус |
| Пул завдань | Нове завдання з'явилося в пулі вашої ролі/відділу |
| Завершення процесу | Процес, ініціатором якого ви були, завершено |
| Таймер | Дедлайн процесного завдання наближається |
| Запрошення | Вас запросили в організацію |
| Коментар | Хтось залишив коментар у вашому завданні |

### Налаштування каналів

1. Для кожного типу події ви можете увімкнути або вимкнути два канали:
   - **In-app** — сповіщення у додатку (дзвіночок). За замовчуванням **увімкнено**.
   - **Email** — сповіщення на електронну пошту. За замовчуванням **вимкнено** (opt-in).
2. Перемикайте тумблери навпроти кожного типу події.
3. Зміни зберігаються автоматично.

### Стани за замовчуванням

- Якщо ви не налаштовували уподобання, діють значення за замовчуванням:
  - **In-app**: увімкнено для всіх подій (opt-out — ви вимикаєте те, що не потрібно).
  - **Email**: вимкнено для всіх подій (opt-in — ви вмикаєте те, що потрібно).

## Поради

- Увімкніть email-сповіщення для критичних подій (призначення завдань, дедлайни), щоб не пропустити важливе.
- Вимкніть in-app сповіщення для подій, що генерують багато повідомлень (наприклад, коментарі), якщо вони вас відволікають.
- Налаштування індивідуальні — вони не впливають на інших учасників організації.
- Якщо ви не отримуєте email-сповіщення, перевірте папку «Спам» та правильність email-адреси в профілі.
