## Огляд

Після запуску процесу ви можете відстежувати його прогрес у реальному часі. Сторінка деталей екземпляра показує поточний етап, активні токени, дедлайни таймерів та повну історію виконання. Також доступне скасування процесу.

## Кроки

### Сторінка деталей екземпляра

1. Перейдіть до розділу **«Процеси»** → список екземплярів.
2. Натисніть на потрібний екземпляр.
3. На сторінці деталей відображається:
   - **Статус** — активний (running), завершений (completed), скасований (cancelled).
   - **Path Stepper** — візуальний покроковий індикатор пройдених та поточних етапів.
   - **Контекст** — змінні процесу (ProcessContextCard).
   - **Поточні токени** — які вузли зараз активні.

### Path Stepper (MyPathStepper)

1. Відображає послідовність пройдених етапів від старту до поточного.
2. Пройдені етапи позначені зеленим, поточний — синім, майбутні — сірим.
3. Показує, який шлях пройшов процес через XOR-розгалуження.

### Таймери та дедлайни

1. Якщо етап має **таймер**, на картці відображається бейдж із часом до спрацювання.
2. **Прострочений** таймер позначається червоним індикатором.
3. Таймер показується тільки для активних (waiting) токенів — завершені та скасовані не відображають бейджі.
4. Таймери можуть автоматично переводити процес на наступний етап або скасовувати завдання.

### Фільтрація списку екземплярів

1. На сторінці списку процесів можна фільтрувати за:
   - **Статусом** (активний, завершений, скасований).
   - **Визначенням процесу**.
   - **Датою запуску**.
2. Список підтримує пагінацію та сортування.

### Скасування процесу

1. На сторінці деталей екземпляра натисніть **«Скасувати процес»**.
2. Підтвердіть дію у діалозі підтвердження.
3. Усі активні токени будуть зупинені, пов'язані завдання скасовані.
4. Скасований процес не можна відновити.

## Поради

- Path Stepper допомагає зрозуміти, який маршрут обрав процес на розгалуженнях.
- Контекст процесу містить усі дані, введені у формах на попередніх етапах.
- Прострочені таймери видно на процесній дошці — зверніть увагу на червоні індикатори.
- Скасування процесу доступне тільки для активних екземплярів.
