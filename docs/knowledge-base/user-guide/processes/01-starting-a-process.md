## Огляд

Бізнес-процеси в Procivo працюють за моделлю BPMN: спершу адміністратор створює та публікує визначення процесу (process definition), а потім користувачі запускають його екземпляри. Кожен запуск створює окрему копію процесу зі своїми даними та прогресом.

## Кроки

### Вибір процесу для запуску

1. Перейдіть до розділу **«Процеси»** у бічній панелі.
2. На сторінці процесів ви побачите список **опублікованих визначень процесів**.
3. Кожне визначення показує: назву, опис, версію, кількість активних екземплярів.
4. Натисніть кнопку **«Запустити»** біля потрібного визначення.

### Стартова форма

1. Якщо процес має стартову форму, з'явиться діалог із полями для заповнення.
2. Заповніть обов'язкові поля (позначені зірочкою).
3. Натисніть **«Запустити процес»**.
4. Якщо стартової форми немає, процес запуститься одразу після натискання «Запустити».

### Перегляд екземпляра процесу

1. Після запуску вас буде перенаправлено на сторінку деталей екземпляра.
2. На сторінці відображається:
   - **Статус** процесу (активний, завершений, скасований).
   - **Поточний етап** — на якому вузлі BPMN знаходиться виконання.
   - **Контекст** — змінні та дані процесу.
3. Список усіх ваших екземплярів доступний на сторінці процесів.

## Поради

- Запустити можна тільки **опублікований** процес — чернетки недоступні для запуску.
- Кожен запуск створює новий незалежний екземпляр — зміни в одному не впливають на інші.
- Стартова форма може містити різні типи полів: текст, числа, дати, вибір зі списку.
- Після запуску процес автоматично створює завдання для відповідальних учасників.
