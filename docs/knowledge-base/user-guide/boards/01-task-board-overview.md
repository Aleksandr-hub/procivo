## Огляд

Дошки — це візуальний інструмент управління роботою в Procivo. Система підтримує два типи дошок: дошка завдань (Task Board) для канбан-управління та процесна дошка (Process Board) для відстеження бізнес-процесів як пайплайну.

## Кроки

### Типи дошок

| Тип | Призначення | Колонки |
|-----|------------|---------|
| **Дошка завдань** | Канбан для управління завданнями | За статусами завдань |
| **Процесна дошка** | Пайплайн для процесів | За етапами BPMN-процесу |

### Створення дошки

1. Перейдіть до розділу **«Дошки»** у бічній панелі.
2. Натисніть кнопку **«Створити дошку»**.
3. Оберіть тип дошки:
   - **Дошка завдань** — для ручних завдань з канбан-підходом.
   - **Процесна дошка** — для візуалізації етапів конкретного бізнес-процесу.
4. Введіть назву дошки.
5. Для процесної дошки оберіть **опубліковане визначення процесу**.
6. Натисніть **«Створити»**.

### Список дошок

1. На сторінці дошок відображаються всі створені дошки організації.
2. Кожна дошка показує: назву, тип, кількість карток.
3. Натисніть на дошку для переходу до неї.

### Структура картки

Картка на дошці відображає ключову інформацію:
- **Аватар виконавця** — фото або ініціали.
- **Чіп пріоритету** — кольоровий індикатор (критичний, високий, середній, низький).
- **Мітки** — до 3 міток з кольорами, решта як «+N».
- **Дедлайн** — дата та час, позначається червоним при простроченні.
- **Кількість коментарів** — іконка з числом.

## Поради

- Використовуйте дошку завдань для повсякденного управління роботою команди.
- Використовуйте процесну дошку для моніторингу бізнес-процесів як пайплайну.
- Одна організація може мати необмежену кількість дошок різних типів.
- Картки на дошках інтерактивні — натисніть для переходу до деталей.
