## Огляд

Свімлейни та фільтри допомагають зосередитися на потрібних завданнях на дошці. Свімлейни групують картки по горизонталі (за виконавцем або пріоритетом), а фільтри дозволяють швидко знайти конкретні завдання за текстом, мітками, датою або виконавцем.

## Кроки

### Свімлейни (Swimlanes)

1. На дошці натисніть кнопку **«Свімлейни»** у панелі керування.
2. Оберіть режим групування:
   - **За виконавцем** — кожна горизонтальна смуга відповідає одному виконавцю.
   - **За пріоритетом** — смуги для кожного рівня пріоритету.
   - **Без свімлейнів** — стандартний вигляд дошки.
3. Картки автоматично розподіляться по відповідних свімлейнах.
4. Кожен свімлейн можна згорнути/розгорнути натисканням на заголовок.

### Quick Filter Bar (Панель швидких фільтрів)

1. Над дошкою розташована **панель швидких фільтрів**.
2. Доступні фільтри:
   - **Текстовий пошук** — введіть частину назви або опису завдання.
   - **Виконавець** — оберіть одного або кількох виконавців.
   - **Мітки** — оберіть мітки для фільтрації.
   - **Діапазон дат** — фільтрація за дедлайном (від — до).
3. Фільтри застосовуються миттєво, без перезавантаження сторінки.
4. Можна комбінувати кілька фільтрів одночасно.

### Збереження фільтрів через URL

1. Усі активні фільтри автоматично зберігаються в **URL-параметрах** сторінки.
2. Скопіюйте URL та поділіться ним з колегами — вони побачать ту ж дошку з тими ж фільтрами.
3. При скиданні фільтра відповідний параметр видаляється з URL.

## Поради

- Свімлейни «За виконавцем» ідеальні для щоденних стендапів — кожен бачить свої завдання.
- Свімлейни «За пріоритетом» допомагають зосередитися на критичних завданнях.
- URL-параметри зберігаються при оновленні сторінки — ваші фільтри не зникнуть.
- Текстовий пошук працює по назві та опису завдання.
- Для очищення всіх фільтрів натисніть кнопку **«Скинути»** на панелі фільтрів.
