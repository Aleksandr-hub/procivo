## Огляд

Процесна дошка як пайплайн — це розширений вигляд, де кожна колонка відповідає етапу BPMN-процесу, а кожна картка — це окремий екземпляр процесу. Цей формат ідеальний для менеджерів, які відстежують потік роботи через усі етапи від початку до завершення.

## Кроки

### Структура пайплайну

1. **Колонки** = етапи BPMN-процесу, розташовані зліва направо у топологічному порядку.
2. **Картки** = екземпляри процесу, розташовані у колонці поточного активного етапу.
3. Порядок колонок визначається автоматично на основі BPMN-схеми (алгоритм Kahn BFS).

### Робота з пайплайном

1. Перегляньте, скільки екземплярів знаходиться на кожному етапі — це видно по кількості карток та числу в заголовку колонки.
2. **Drag-to-Complete**: перетягніть картку в наступну колонку для завершення поточного етапу:
   - Якщо перехід має форму, відкриється **ActionFormDialog** для введення даних.
   - Якщо перехід без форми, екземпляр автоматично перейде на наступний етап.
3. Натисніть на картку для переходу до деталей екземпляра процесу.

### Метрики в заголовку

1. У верхній частині пайплайну відображаються ключові метрики:
   - **Активні екземпляри** — загальна кількість процесів у роботі.
   - **Sparkline** — мініатюрний графік завершень за останні дні.
2. Кожна колонка показує кількість екземплярів на цьому етапі.

### Порівняння з процесною дошкою

Процесна дошка як пайплайн — це розширення базової процесної дошки з акцентом на:
- Візуалізацію потоку зліва направо.
- Метрики пропускної здатності.
- Drag-to-complete з інтеграцією ActionFormDialog.

## Поради

- Пайплайн особливо корисний для процесів з чітким лінійним потоком (наприклад, погодження, онбордінг).
- Для процесів з багатьма паралельними гілками пайплайн може мати багато колонок — фокусуйтесь на ключових етапах.
- Drag-to-complete працює через Workflow Engine — усі бізнес-правила та XOR-маршрутизація зберігаються.
- Sparkline у заголовку допомагає оцінити тренд завершення процесів без переходу до аналітики.
- Об'єднуйте пайплайн з фільтрами для фокусування на конкретних екземплярах.
