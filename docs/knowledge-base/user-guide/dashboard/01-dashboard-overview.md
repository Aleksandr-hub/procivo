## Огляд

Дашборд — це головна сторінка Procivo, яка дає повну картину вашої роботи. Тут зібрані KPI-картки з ключовими показниками, графіки по завданнях, список поточних завдань та стрічка останніх подій в організації.

## Кроки

### KPI-картки

1. У верхній частині дашборду розміщені картки з ключовими показниками:
   - **Всього завдань** — загальна кількість завдань у вашій організації.
   - **Виконано** — кількість завершених завдань.
   - **Відсоток виконання** — частка виконаних завдань від загальної кількості.
   - **Активні процеси** — кількість запущених бізнес-процесів.
2. Кожна картка містить мініатюрний графік (sparkline) для візуалізації тренду.

### Мої завдання

1. Віджет **«Мої завдання»** показує завдання, призначені на вас.
2. Завдання групуються за категоріями:
   - **Прострочені** — завдання з дедлайном, що вже минув.
   - **Сьогодні** — завдання з дедлайном на сьогодні.
   - **Найближчі** — завдання з дедлайном у найближчі дні.
3. Натисніть на завдання для переходу до його деталей.

### Графіки

1. **Кільцева діаграма (Donut)** — розподіл завдань за статусами (нове, в роботі, виконано, скасовано).
2. **Лінійний графік** — кількість виконаних завдань за останні дні.
3. Кольори графіків адаптуються до теми інтерфейсу (світла/темна).

### Стрічка активності

1. У нижній частині дашборду розташована стрічка **останніх подій** з аудит-логу.
2. Відображає створення, зміну статусу та завершення завдань і процесів.
3. Кожен запис містить автора дії, тип події, час та посилання на об'єкт.

## Поради

- Дашборд автоматично завантажує дані при вході — усі віджети оновлюються паралельно.
- Якщо один із віджетів не завантажився, решта продовжать працювати (ізольовані запити).
- Відсоток виконання обчислюється на стороні клієнта з даних розподілу за статусами.
- Стрічка активності працює як компонент аудит-логу з фільтрацією по організації.
