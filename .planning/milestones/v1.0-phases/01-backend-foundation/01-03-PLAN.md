**Step 1: Add lint() method to ExpressionEvaluator**

Add a `lint` method to ExpressionEvaluator that validates expression syntax at design time:
```php
use Symfony\Component\ExpressionLanguage\Parser;

/**
 * Validate expression syntax at design time.
 * Does NOT evaluate — only checks if the expression is syntactically valid.
 * Uses IGNORE_UNKNOWN_VARIABLES since runtime variables are unknown at design time.
 *
 * @throws SyntaxError if expression is syntactically invalid
 */
public function lint(string $expression): void
{
    $this->expressionLanguage->lint(
        $expression,
        [],
        Parser::IGNORE_UNKNOWN_VARIABLES,
    );
}
```

**Step 3: Add expression validation to ProcessGraphValidator**

Inject ExpressionEvaluator into ProcessGraphValidator constructor:
```php
public function __construct(
    private readonly ExpressionEvaluator $expressionEvaluator,
) {}
```

Add a new private validation method `validateExpressionSyntax` that is called from `validate()`:
```php
private function validateExpressionSyntax(array $transitions, array &$errors): void
{
    foreach ($transitions as $transition) {
        $expression = $transition->conditionExpression();
        if (null === $expression || '' === $expression->value()) {
            continue;
        }

        try {
            $this->expressionEvaluator->lint($expression->value());
        } catch (SyntaxError $e) {
            $errors[] = sprintf(
                'Transition "%s": invalid expression syntax — %s',
                $transition->name() ?? $transition->id()->value(),
                $e->getMessage(),
            );
        }
    }
}
```

Call `$this->validateExpressionSyntax($transitions, $errors);` in the `validate()` method, after existing validations.
