```php
#[AsMessageHandler(bus: 'query.bus')]
final readonly class ListVersionsHandler
{
    public function __construct(
        private ProcessDefinitionVersionRepositoryInterface $versionRepository,
    ) {}

    /**
     * @return list<ProcessDefinitionVersionDTO>
     */
    public function __invoke(ListVersionsQuery $query): array
    {
        $defId = ProcessDefinitionId::fromString($query->processDefinitionId);
        $versions = $this->versionRepository->findByProcessDefinitionId($defId);

        // Sort by version number descending (newest first)
        usort($versions, static fn ($a, $b) => $b->versionNumber() <=> $a->versionNumber());

        return array_map(
            static fn (ProcessDefinitionVersion $v) => ProcessDefinitionVersionDTO::fromEntity($v),
            $versions,
        );
    }
}
```

**MigrateProcessInstancesHandler.php — create handler:**

This handler migrates ALL running process instances of a given definition to a target version. It updates the `version_id` column in the **read model** table `workflow_process_instances_view`. The event store (`workflow_process_events`) is immutable and is NOT modified.

The handler:
1. Loads the target version to verify it exists
2. Loads the source version (currently used by running instances) to validate schema compatibility
3. Updates `version_id` in the read model for all running instances of this definition

```php
#[AsMessageHandler(bus: 'command.bus')]
final readonly class MigrateProcessInstancesHandler
{
    public function __construct(
        private ProcessDefinitionVersionRepositoryInterface $versionRepository,
        private ProcessDefinitionRepositoryInterface $definitionRepository,
        private Connection $connection,
    ) {}

    public function __invoke(MigrateProcessInstancesCommand $command): void
    {
        $targetVersion = $this->versionRepository->findById(
            ProcessDefinitionVersionId::fromString($command->targetVersionId),
        );

        if (null === $targetVersion) {
            throw new \DomainException('Target version not found.');
        }

        // Validate that target version belongs to the specified definition
        if ($targetVersion->processDefinitionId()->value() !== $command->processDefinitionId) {
            throw new \DomainException('Target version does not belong to the specified process definition.');
        }

        // Schema compatibility validation: verify that all task node IDs
        // present in currently running instances also exist in the target version.
        // This prevents migrating to a version that removed nodes where tokens are waiting.
        $targetGraph = ProcessGraph::fromSnapshot($targetVersion->nodesSnapshot());
        $targetNodeIds = array_map(
            static fn (array $node): string => $node['id'],
            $targetVersion->nodesSnapshot(),
        );

        // Find active token node_ids for running instances of this definition
        $runningRows = $this->connection->fetchAllAssociative(
            'SELECT id, tokens FROM workflow_process_instances_view WHERE definition_id = :definitionId AND status = :status',
            ['definitionId' => $command->processDefinitionId, 'status' => 'running'],
        );

        foreach ($runningRows as $row) {
            $tokens = json_decode((string) $row['tokens'], true, 512, \JSON_THROW_ON_ERROR);
            foreach ($tokens as $token) {
                if (isset($token['node_id'], $token['status']) && 'completed' !== $token['status']) {
                    if (!\in_array($token['node_id'], $targetNodeIds, true)) {
                        throw new \DomainException(\sprintf(
                            'Cannot migrate instance %s: active token at node "%s" which does not exist in target version %d.',
                            $row['id'],
                            $token['node_id'],
                            $targetVersion->versionNumber(),
                        ));
                    }
                }
            }
        }

        // Update version_id in the read model for all running instances of this definition.
        // Read model table: workflow_process_instances_view
        // Columns: id (VARCHAR 36), definition_id (VARCHAR 36), version_id (VARCHAR 36), status (VARCHAR 20)
        $affected = $this->connection->executeStatement(
            'UPDATE workflow_process_instances_view SET version_id = :targetVersionId WHERE definition_id = :definitionId AND status = :status',
            [
                'targetVersionId' => $command->targetVersionId,
                'definitionId' => $command->processDefinitionId,
                'status' => 'running',
            ],
        );
    }
}
```

**Key architectural note:** The event store table `workflow_process_events` (columns: `id`, `aggregate_id`, `event_type`, `payload` JSON, `version`, `occurred_at`) is immutable — events are never modified. The `ProcessStartedEvent` payload contains `version_id` as the original version. Migration only updates the read model projection (`workflow_process_instances_view.version_id`). When the process engine needs the graph for token advancement, it reads `version_id` from the read model view, not from the event store directly.
