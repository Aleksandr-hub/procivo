## Decisions Made

- **executeAction via workflow-action endpoint:** Uses POST `/tasks/{taskId}/workflow-action` not `/transition` — ensures BPMN token state machine advances correctly, not task FSM
- **ActionFormDialog reuse:** Existing ActionFormDialog accepts `StatusAction` prop; converted `WorkflowActionDTO` to `StatusAction` shape in the onDrop handler
- **Drag prevention on no active task:** When `instance.activeTaskId` is null, `isDraggable` computed returns false and the card has `draggable="false"` + CSS `cursor: pointer` to signal non-draggable state
