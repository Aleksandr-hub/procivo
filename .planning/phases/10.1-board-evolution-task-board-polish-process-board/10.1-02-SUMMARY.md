- swimlanes computed always returns array — `none` mode returns single `{ key: 'all' }` lane — template v-for works uniformly
- assigneeOptionsWithAll prepends empty-value option inside QuickFilterBar, parent passes clean options without All entry
- Priority swimlane shows all 4 rows (including empty) — consistent layout regardless of filter state
- URL param sync: `undefined` removes param from URL, never empty string — clean shareable URLs
